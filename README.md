💰 Bank Account Management System

This project is a simple **Bank Account Management System** developed while learning **Object-Oriented Programming (OOP)** concepts in JavaScript.

It demonstrates core OOP principles like:
- Classes & Objects
- Constructors
- Inheritance
- Method Overriding

 🚀 Features

- Create bank account objects
- Deposit money
- Withdraw money
- Check account balance
- Demonstrates inheritance and overriding concepts


🧠 Concepts Used

This project was mainly built for learning and understanding:

- Classes & Objects
- Constructors
- Inheritance
- Polymorphism (Method Overriding)
